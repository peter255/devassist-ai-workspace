# Screenshots

Live screenshots captured from DevAssist AI Workspace running against Azure AI services.

## Available screenshots

| File | Module | Description |
|------|--------|-------------|
| `dashboard.png` | Dashboard | Workspace overview — document count, indexed count, recent analyses |
| `copilot-arabic-answer.png` | Knowledge Copilot | Arabic question with RTL-formatted AI answer and source citations |
| `copilot-english-answer.png` | Knowledge Copilot | English question with grounded answer citing indexed markdown docs |
| `ticket-analyzer.png` | Ticket Analyzer | Structured triage result — severity HIGH, category, impacted module, suggested action |
| `requirement-breakdown.png` | Requirement Breakdown | Full breakdown: functional summary, backend/frontend tasks, testing checklist, risks, assumptions, acceptance criteria |

## How to add screenshots

1. Right-click each screenshot in the Cursor chat and **Save image as…**
2. Save it to this folder (`docs/screenshots/`) using the filename from the table above.
3. Or take a fresh screenshot with `Windows + Shift + S` and save with the same name.
